POST /download https://movieapi-1q90.onrender.com/api/v1/content/download
Initiates a background download job for a specific video. Returns a download ID for tracking progress.
ParameterTypeRequiredDescription
urlstringRequiredThe content URL to download
variantstringOptionalAudio language variant (e.g., "Latino", "Spanish")
preferredServerstringOptionalPreferred server name (e.g., "streamwish")
 
POST https://movieapi-1q90.onrender.com/api/v1/content/download
Content-Type: application/json

{
  "url": "https://www.pelisplushd.la/movie/avatar-2009",
  "variant": "Latino",
  "preferredServer": "streamwish"
}
Response Example
{
  "success": true,
  "data": {
    "downloadId": "dl_abc123xyz",
    "status": "queued",
    "message": "Download started successfully"
  }
}
GET /download/:id https://movieapi-1q90.onrender.com/api/v1/content/download/dl_abc123xyz
Retrieves the current status and progress of a background download job.
ParameterTypeRequiredDescription
idstringRequiredThe download ID returned from the POST /download endpoint
 
GET https://movieapi-1q90.onrender.com/api/v1/content/download/dl_abc123xyz
Response Example
{
  "success": true,
  "data": {
    "id": "dl_abc123xyz",
    "status": "downloading",
    "progress": 45,
    "currentServer": "streamwish",
    "fileSize": "524288000"
  }
}
Status Values
• "queued"     - Waiting to start
• "preparing"  - Resolving download URLs
• "downloading" - Actively downloading
• "completed"  - Download finished
• "failed"     - Download failed
POST /batch https://movieapi-1q90.onrender.com/api/v1/content/batch
Initiates batch downloads for multiple episodes of a TV series or anime. Returns an array of download IDs.
ParameterTypeRequiredDescription
mediaUrlstringRequiredThe series content URL
seasonintegerRequiredSeason number to download
episodesarrayRequiredArray of episode numbers to download
variantstringOptionalAudio language variant (e.g., "Latino")
 
POST https://movieapi-1q90.onrender.com/api/v1/content/batch
Content-Type: application/json

{
  "mediaUrl": "https://www.pelisplushd.la/series/breaking-bad",
  "season": 1,
  "episodes": [1, 2, 3, 4, 5],
  "variant": "Latino"
}
Response Example
{
  "success": true,
  "data": {
    "batchId": "batch_xyz789",
    "downloads": [
      { "episode": 1, "downloadId": "dl_ep1_abc" },
      { "episode": 2, "downloadId": "dl_ep2_def" },
      { "episode": 3, "downloadId": "dl_ep3_ghi" }
    ],
    "total": 3,
    "message": "Batch download started"
  }
}
